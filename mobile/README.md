# Mobile iOS Application

iOS application built with SwiftUI for event management and check-in.

## Features

- User authentication
- View active events
- View approved registrations for events
- QR code scanning for check-in/check-out

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Setup

1. Open the project in Xcode
2. Update the API base URL in `Config.swift`
3. Build and run
